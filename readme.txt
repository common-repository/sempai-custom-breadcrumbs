A WordPress plugin that allows you to add custom breadcrumbs to your website with the ability to customize the styling.

== Description ==

Custom Breadcrumbs is a user-friendly WordPress plugin that helps you add customizable breadcrumb navigation to your website. Breadcrumbs improve the user experience and SEO by making it easier for visitors to navigate your site and for search engines to understand its structure.

= Features =
* Easy to install and use.
* Compatible with most WordPress themes.
* Improves site navigation and SEO.
* Supports multiple breadcrumb levels.

== Installation ==

1. Upload the `custom-breadcrumbs` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the WordPress page editor and at the bottom of the page, you will find a block for breadcrumbs where you can configure them as desired.

== Frequently Asked Questions ==

= How do I customize the breadcrumb styling? =
You can customize the breadcrumbs' appearance using custom CSS. After configuring the breadcrumbs, navigate to your theme’s customizer or add your styles directly to the theme's stylesheet.

= Does this plugin improve my website's SEO? =
Yes, breadcrumbs can improve your site's SEO by helping search engines better understand your site structure and hierarchy. It makes navigation easier for both users and crawlers.

= What if my theme already has breadcrumbs? =
If your theme already includes a breadcrumb feature, you can still use this plugin to override the existing breadcrumbs with new ones.
